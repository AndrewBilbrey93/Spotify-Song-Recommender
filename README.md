# Spotify Song Recommender
This project utilizes the Spotify API and K-Means Clustering to recommend songs based on the characteristics of a song input by the user. Users can type in the name of a song, and the system will return similar songs based on clustering techniques.

## Technologies Used
- Python for data analysis and model building
- Spotify API for accessing music data
- Scikit-Learn for K-Means clustering
- Pandas for data manipulation

## Features
- Input a song title to receive recommendations
- Visualize clusters of similar songs based on features such as tempo, danceability, and loudness
- Dynamic suggestions based on user input for enhanced music discovery

## Data Source
The data for this project is sourced from the Spotify API, which provides access to various audio features and metadata for songs.
